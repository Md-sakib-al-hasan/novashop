# 🛍️ NovaShop

**NovaShop** is a premium, high-performance e-commerce platform built with the latest web technologies. It features a stunning, minimalistic design with smooth animations and a powerful admin dashboard.

---

## ✨ Features

- **Professional Auth Flow**: Beautifully designed login and registration pages with glassmorphism and multi-step forms.
- **Dynamic Admin Dashboard**: Real-time analytics, product management, and user oversight.
- **Premium UI/UX**: Smooth transitions and micro-interactions powered by Framer Motion.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop experiences.
- **State-of-the-Art Styling**: Built using the cutting-edge **Tailwind CSS v4**.

---

## 🛠️ Technology Stack

| Component | Technology |
| :--- | :--- |
| **Framework** | [Next.js 16](https://nextjs.org/) (App Router) |
| **Styling** | **[Tailwind CSS v4](https://tailwindcss.com/)** |
| **Animations** | [Framer Motion](https://www.framer.com/motion/) |
| **UI Components** | [Shadcn UI](https://ui.shadcn.com/) (Radix UI) |
| **State Management** | React Context (AuthContext) |
| **Form Handling** | React Hook Form & Zod |
| **Icons** | Lucide React |

## 🚀 Getting Started

### Prerequisites

- Node.js 18+ 
- npm / pnpm / yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd novashop
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open the app:**
   Navigate to [http://localhost:3000](http://localhost:3000) in your browser.

## 📂 Project Navigation

To explore every part of the project, you can navigate to these routes:

- **Customer View**:
  - ` / ` - Home / Landing Page
  - ` /login ` - Login Page
  - ` /register ` - Registration Page
  - ` /shop ` - Product Browsing

- **Admin View**:
  - ` /admin ` - Dashboard Overview
  - ` /admin/products ` - Inventory Management
  - ` /admin/analytics ` - Sales & Performance
  - ` /admin/users ` - User Management
  - ` /admin/settings ` - System Configurations

---

## 💡 How to Use

1. **Authentication**: Start by visiting `/register` to see the high-quality multi-step registration flow. 
2. **Browsing**: Explore the `/shop` page for a smooth browsing experience.
3. **Administration**: Access the `/admin` panel to manage the entire shop's backend from a sleek interface.
